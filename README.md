# Computer Society of Kirinyaga
## Front end repository for the CSK web app.
```React```, ```Material UI```, ```Redux```, ```React-router-DOM```

## Getting Started
1. You need to have npm installed in your system.
1. If you don't have npm installed in your computer, you can [Learn how to set up npm on your computer.](https://www.youtube.com/results?search_query=how+to+setup+npm)
1. You also need to have git installed in your computer.
1. If you don't have git installed in your computer, you can [Learn how to install and setup git on your computer](https://www.youtube.com/results?search_query=install+and+set+up+git)

If you satisfy all the requirements mentioned above
1. Open your terminal/command prompt/git bash.
1. Run the following command: ```git clone https://github.com/Computer-Society-Of-Kirinyaga/csk-front-end```
1. Navigate into the folder: ```cd csk-front-end```
1. Install the dependencies: ```npm install```
1. Once the installation is complete, spin up the development server to view in the browser using the command ```npm run dev```.
1. CONGRATULATIONS!!!!!.

## Contributing
If you found any bugs, issues or have an idea, you can [open a new issue here](https://github.com/Computer-Society-Of-Kirinyaga/csk-front-end/issues) and let us know what's on your mind.

If you want to contribute to development of a new feature, follow the steps below:
1. Make sure you have the project locally using the steps followed above, it is okay if you want to code using tools like github codespaces.
1. Create a new branch with the branch name made up of the name of the feature you want to create slash your name, preferably github username for example ```navbar/codeme254``` or ```footer/kelcho-spense```.
1. Work on the feature/bug.
1. Push your code and create a new pull request once you are done.
1. THANK YOU!!!

*Please don't attempt to fix a bug without opening a new issue describing the bug.*
